Connect Claude Code

Set up your Agnes workspace in Claude Desktop — no terminal needed. Takes about 2 minutes.

Requires Claude Desktop to be installed
1

Download the setup bundle

Go to AI Cowork and click Download Setup Bundle. Your browser will ask where to save the file.

The bundle expires in 24 hours and is single-use. If it has expired, just download a new one.
2

Unzip the file

Double-click the downloaded agnes-cowork-setup-*.zip file. A folder with the same name appears next to it.

3

Open the folder in Claude Desktop

Open Claude Desktop, then follow this path:

Cowork › New project › Use an existing folder › Select the unzipped folder › Create
4

Agnes connects automatically

The project opens and Agnes sets everything up in the background — no extra steps needed. Start by asking:

What data do I have access to?

Agnes will list your available datasets. From now on, data syncs automatically every time you open the project.

FAQ

The bundle expired or login fails
Each bundle is valid for 24 hours and is single-use. Go to AI Cowork, click Download Setup Bundle again, and repeat the steps above.
How do I revoke an old bundle?
On AI Cowork, expand Active setup bundles under the Connect Claude Code section and click Revoke next to the bundle you want to remove. Bundles also expire automatically after 24 hours.
How do I know Agnes is connected?
Ask Claude: "What data do I have access to?" — it should respond with a list of available tables. If it says it has no access, check your network connection or download a fresh bundle.
Does data sync automatically?
Yes. Every time you open the project in Claude Desktop, Agnes pulls the latest data in the background. You don't need to run anything manually.
Can I use the same bundle on multiple machines?
No — each bundle is single-use. Once it has been exchanged for a session on one machine, it cannot be used again. Download a separate bundle for each machine.
