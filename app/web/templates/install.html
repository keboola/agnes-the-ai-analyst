Install the Agnes CLI

This server: {{ server_url }} (version {{ agnes_version }})

One-liner (Linux / macOS)

curl -fsSL {{ server_url }}/cli/install.sh | bash

Manual

  1. Download: {{ server_url }}/cli/download
  2. Install: 
    uv tool install ./agnes-*.whl
# or
python3 -m pip install --user ./agnes-*.whl
  3. Seed server URL: 
    mkdir -p ~/.config/da
echo "server: {{ server_url }}" > ~/.config/da/config.yaml

Connect

Create a personal access token (see /profile) and export it:

export DA_TOKEN=<your-token>
da auth whoami

Claude Code / MCP

Store your token in ~/.config/da/token.json (Claude Code reads this automatically via the da entrypoint) or export DA_TOKEN in your shell.
