What this is: a ready-made Cowork project folder — your Agnes skills plus a live MCP connection to your data (catalog, schema, query …). Best for working with your data in Cowork. Scope: it only works inside this one project — you open the downloaded folder as a Cowork project and Agnes is wired up there. It does not follow you into your other Cowork projects. To use your skills everywhere, add them as packages below.
Requires Claude Desktop. The bundle expires in 24 hours and is single-use. Setup guide
  1. Click Download Setup Bundle below
  2. Unzip the downloaded file
  3. Open Claude Desktop → Cowork → New project → Use an existing folder → select the unzipped folder
  4. Agnes configures automatically at session start
About skills: your Agnes skills load automatically when you open the folder — Claude can use them in chat right away. They will not appear in Cowork's Customize → Skills list (a Cowork UI limitation, not a problem). To confirm a skill works, just ask Claude to use it — e.g. “use the develop-component skill”.
